ABOUT

The project itself and the team behind Planet Seeker.

The Project

Planet Seeker is an interactive astronomy tool that visualizes the real-time phases of every planet in our solar system as seen from Earth. By querying NASA JPL's Horizons API, the site retrieves live phase angles and solar elongation data for each body, then renders an accurate illumination portrait using HTML5 Canvas.

How It Works

🖥️

HTML5 Canvas

Each planet is drawn by layering a dark circle, a lit semicircle, and a terminator ellipse whose width is controlled by the phase angle.

🛰️

NASA JPL Horizons API

Live ephemeris data fetched on page load — phase angle, solar elongation, and orbital speed per body.

🌐

CORS Proxy

corsproxy.io relays API calls to NASA's servers to get around browser cross-origin restrictions.
